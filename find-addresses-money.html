üéØ Summary - What You Learned

For Addresses:

  1. Use Ctrl + F to find keywords (Twin Oaks, San Diego, Via Roma)
  2. SELECT the entire address block (all lines)
  3. COPY it (Ctrl + C)
  4. Open Find & Replace (Ctrl + H)
  5. PASTE into "Find what" (Ctrl + V)
  6. Type merge field in "Replace with"
  7. Click "Replace All"

For Money Amounts:

  1. Use Ctrl + F to find keywords (rent, deposit, total)
  2. NOTE THE EXACT FORMAT ($25.00 vs $25 vs 25.00)
  3. Include $ sign if it's there
  4. Include commas if they're there ($3,050.00)
  5. Include decimals if they're there (.00)
  6. Open Find & Replace (Ctrl + H)
  7. Type EXACT amount in "Find what"
  8. Type merge field in "Replace with"
  9. Click "Replace All"
