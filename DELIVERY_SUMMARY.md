## How It Works

### For the User (Simple):
1. Open `index.html` in browser
2. Click "Start New Contract"
3. Fill in 18 questions (one at a time)
4. Review answers
5. Click "Generate PDF"
6. PDF downloads automatically

### Behind the Scenes (Professional):
1. Form validates all inputs
2. Data stored in memory
3. jsPDF creates new PDF document
4. Adds header with RomaRentals branding
5. Formats all text with proper fonts/sizes
6. Adds all 19 clauses with correct spacing
7. Calculates total rent automatically
8. Adds signature lines
9. Adds additional documentation space
10. Downloads PDF to user's computer
